# Aplicativo ativo
Abrir um site (offline-local) para rodar o app (server side on local).
Rodar via terminal. Anotar todas dependencias no README.md
$ make
O usuário vai usar apartir do site, com algumas funcionalidades, tudo na mesma pagina

Botão para Gravar amostra de perfil (10seg) # usar como padrão a voz da demo
Botão para live, apartir do audio do microfone vai filtrar com base na voz do perfil e devolver na saida de audio.
Botão deslizante 1-256 para "termometro" da voz, aumentando ou diminuindo o filtro.

O site vai escutar realtime, filtrar a voz, e transmitir no fone a voz sem ruído.

# Correções no filtro do audio. 
Precisa cortar o Perfil, ou seja 10 seg de audio no maximo.
Precisa ser ajustavel no live, (pode ter multiplos ajustes manuais, pelo usuário)
Usar todos os audios em mp3.

Como nós vamos resolver: conceito.
