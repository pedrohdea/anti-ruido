# Design System

> Inspirado em portfolios editoriais minimalistas.

---

# Princípios

O design deve ser praticamente invisível.

A interface existe apenas para apresentar o conteúdo.

Todo elemento deve possuir um propósito claro.

Priorizar:

- Espaço em branco
- Tipografia
- Hierarquia visual
- Simplicidade

Evitar:

- Excesso de cores
- Gradientes
- Sombras pesadas
- Bordas chamativas
- Ícones desnecessários
- Ornamentação

---

# Estilo

**Minimalismo Editorial**

Características:

- Clean
- Premium
- Elegante
- Atemporal
- Moderno
- Sofisticado
- Silencioso
- Arquitetônico

Referências:

- Portfólios de arquitetura
- Design escandinavo
- Swiss Design
- Galerias de arte

## Estados

Utilizar mudanças sutis de opacidade.

Exemplo:

opacity: 80%

Disabled:
opacity: 40%

# Grid

Preferência por:

- 2 colunas
- 3 colunas
- Grid assimétrico

As imagens devem respirar.

Nunca deixar elementos "colados".

---

# Bordas

Radius:

```
0px
```

ou

```
4px
```

Nunca utilizar bordas arredondadas exageradas.

---

---

# Cards

Os cards não devem parecer cards.

Evitar:

- sombras
- bordas grossas

Preferir:

- espaçamento
- tipografia

---

# Imagens

As imagens são o elemento principal da interface.

Regras:

- Alta resolução
- Grande destaque
- Poucos elementos ao redor
- Muito espaço em branco

Evitar molduras pesadas.

---

# Ícones

Preferência por:

- outline
- stroke fino
- monocromáticos

Tamanho:

```
20px
```

ou

```
24px
```

---

# Animações

Muito discretas.

Duração:

```
150ms
```

ou

```
200ms
```

Easing:

```
ease-out
```

Evitar:

- bounce
- zoom exagerado
- animações chamativas

```

```

```

```

- aumentar espaçamento vertical
- manter imagens grandes

---

# Filosofia

O usuário deve perceber o conteúdo antes da interface.

O design deve transmitir calma, organização e sofisticação através da simplicidade, utilizando poucos elementos, tipografia refinada e amplo espaço em branco. Cada componente deve contribuir para uma experiência visual leve, discreta e atemporal.
